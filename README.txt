Deverá primeiro ter um servidor TOMCAT8.5 criado.
Para executar o projeto será necessário primeiro importar o ficheiro SdFinal.war
Em seguida inicializar os servidores RMI em remoteImpl.java
Por fim inicializar a página login.jsp
Nota: 1ºvez o programa inicializa deverá acessar ao login com o BI:00000 e password:00000 que estão hardcoded onde depois poderá na opção registar utilizadores criar mais "webadmins" ao escolher o tipo 4
Deverá também inicializar uma consola.java para criar departamentos pois essa opção não se encontrava na checklist pelo que a opcão nao foi incluida na consola web
Caso queira ver resultados de uma eleição ou ver local de voto é necessário seguir os seguintes passos para que nao lhe seja retornado uma pagina em branco depois do submit
1ºCriar uma eleicao com uma data que dê tempo para fazer voto antecipado
2º fazer voto antecipado com um utilizador previamente criado e elegivel
3ºdeixa a eleicao acabar

as porta preconfiguradas sao 12345 para os servidoresRMI
10000 para a porta udp
localhost para ips
